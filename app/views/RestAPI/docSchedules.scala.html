Devoxx API

Documentation for "schedules"

Returns a JSON Array of Links to each schedule. Each conference has one schedule per day.

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showAllSchedules(Conference.devoxxFrance2014.eventCode).absoluteURL()
