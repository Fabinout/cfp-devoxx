@Messages("shortName") API

Documentation for "schedules"

Returns a JSON Array of Links to each schedule. Each conference has one schedule per day.

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showAllSchedules(Conference.currentConference.eventCode).absoluteURL()

Result:

{
    "links": [
        {
            "href": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/conferences/@ConferenceDescriptor.current().eventCode/schedules/wednesday/",
            "rel": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/profile/schedule",
            "title": "Schedule for Wednesday 16th April 2014"
        },
        {
            "href": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/conferences/@ConferenceDescriptor.current().eventCode/schedules/thursday/",
            "rel": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/profile/schedule",
            "title": "Schedule for Thursday 17th April 2014"
        },
        {
            "href": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/conferences/@ConferenceDescriptor.current().eventCode/schedules/friday/",
            "rel": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/profile/schedule",
            "title": "Schedule for Friday 18th April 2014"
        }
    ]
}
}
