Devoxx API

Documentation for "schedules"

Returns a JSON Array of Links to each schedule. Each conference has one schedule per day.

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showAllSchedules(Conference.devoxxFrance2014.eventCode).absoluteURL()

Result:

{
    "links": [
        {
            "href": "http://@confDesc.cfpHostname/api/conferences/@confDesc.eventCode/schedules/wednesday/",
            "rel": "http://@confDesc.cfpHostname/api/profile/schedule",
            "title": "Schedule for Wednesday 16th April 2014"
        },
        {
            "href": "http://@confDesc.cfpHostname/api/conferences/@confDesc.eventCode/schedules/thursday/",
            "rel": "http://@confDesc.cfpHostname/api/profile/schedule",
            "title": "Schedule for Thursday 17th April 2014"
        },
        {
            "href": "http://@confDesc.cfpHostname/api/conferences/@confDesc.eventCode/schedules/friday/",
            "rel": "http://@confDesc.cfpHostname/api/profile/schedule",
            "title": "Schedule for Friday 18th April 2014"
        }
    ]
}
