Messages("shortName") API

Documentation for "conference"

Returns a JSON Array of Link object.

A conference is described by an eventCode, a label and a list of links to extra resources such as the list of speakers and the list of schedules.

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showConference(Conference.currentConference.eventCode).absoluteURL()

Result:

{
    "eventCode": "@confDesc.eventCode",
    "label": "@Messages("longYearlyName"), @Messages(confDesc.timing.datesI18nKey)",
    "links": [
        {
            "href": "http://@confDesc.conferenceUrls.cfpHostname/api/conferences/@confDesc.eventCode/speakers",
            "rel": "http://@confDesc.conferenceUrls.cfpHostname/api/profile/list-of-speakers",
            "title": "See all speakers"
        },
        {
            "href": "http://@confDesc.conferenceUrls.cfpHostname/api/conferences/@confDesc.eventCode/schedules/",
            "rel": "http://@confDesc.conferenceUrls.cfpHostname/api/profile/schedules",
            "title": "See the whole agenda"
        }
    ]
}
}
