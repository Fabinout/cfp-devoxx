Documentation for "conference"

Returns a JSON Array of Link object.

A conference is described by an eventCode, a label and a list of links to extra resources such as the list of speakers and the list of schedules.

Sample:

curl -H "Accept: application/json" @routes.RestAPI.showConference(Conference.currentConference.eventCode).absoluteURL()

Result:

{
    "eventCode": "@ConferenceDescriptor.current().eventCode",
    "label": "@Messages("longYearlyName"), @Messages(ConferenceDescriptor.current().timing.datesI18nKey)",
    "links": [
        {
            "href": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/conferences/@ConferenceDescriptor.current().eventCode/speakers",
            "rel": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/profile/list-of-speakers",
            "title": "See all speakers"
        },
        {
            "href": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/conferences/@ConferenceDescriptor.current().eventCode/schedules/",
            "rel": "http://@ConferenceDescriptor.current().conferenceUrls.cfpHostname/api/profile/schedules",
            "title": "See the whole agenda"
        }
    ]
}
}
