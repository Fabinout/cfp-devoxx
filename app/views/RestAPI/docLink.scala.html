Devoxx API

Documentation for "links"

The "Link" profile object is a Devoxx object that defines a relationship between 2 resources.

  • Href is an absolute URL, that a client may follow to load this resource's content.
  • Rel is an absolute URL to a Profile definition for the target resource designed by href.
  • Title is the link title.

Sample:

{
    "link":
        {
            "href": "http://app-7025f873-e3d4-4c71-b8d2-3f504d2ca2b8.cleverapps.io/api/conferences/devoxxFR2014",
            "rel": "http://app-7025f873-e3d4-4c71-b8d2-3f504d2ca2b8.cleverapps.io/api/profile/conference",
            "title": "See more details about Devoxx France 2014"
        }
}
}
